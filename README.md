# Miami Academy Website

## Overview
Miami Academy is a simple static website created using HTML, CSS, and Bootstrap. The project presents a basic multi-page layout for an academy, including navigation, content sections, and user interface design.

## Tech Stack
- HTML5  
- CSS3  
- Bootstrap  

## Features
- Multi-page website structure (Home, About Us, Courses, Contact)
- Responsive design using Bootstrap
- Navigation between pages
- Basic layout and styling with CSS

## Project Structure
- `index.html` – main homepage  
- `aboutus.html` – information about the academy  
- `course.html` – course offerings  
- `contact.html` – contact page  
- `css/` – stylesheets  
- `pictures/` – images used in the website
